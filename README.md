# Employee_Retention_Analysis
This project analyzes employee retention using machine learning to identify factors that influence whether employees stay or leave an organization. The model achieves an accuracy of 77.93%, providing actionable insights for HR teams and management.

Features:
• Exploratory Data Analysis (EDA) to uncover patterns in employee behavior.
• Feature engineering for improved prediction of retention.
• Machine learning model trained to predict employee turnover.
• Clear evaluation metrics and visualizations for interpretability.
• A reproducible workflow for testing and future improvements.

Dataset:
• The dataset includes employee demographic information, job roles, experience, and performance metrics.
• Data preprocessing steps include handling missing values, encoding categorical variables, and scaling numerical features.
